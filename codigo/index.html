Información

Para utilizar la máquina codificadora deberá elegir
un tipo de codificación.

Tipo Three code

En este modo lo que sucederá es que a cada caracter/letra
que ingrese será cambiado por el 3er caracter/letra que le siga.
Para decodificar sucede lo mismo pero a la inversa, se le restaran 3.

Tipo Key code

En este modo lo que sucederá es que a cada caracter/letra
que ingrese será cambiado por el numero ingresado.
Si usted ingresa la key "123" y el texto "hola",
la "h" pasará a ser la siguiente letra, a la "o"
serán dos letras adelante, la "l" serán 3, y con la "a"
sera una adelante, repitiendo la key hasta el final del texto
Para decodificar sucede lo mismo pero a la inversa.
